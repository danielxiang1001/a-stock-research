# A股大势研判方法论 - 华泰策略框架

> 基于华泰证券"盈利周期（分子）+ 估值资金（分母）+ 多周期共振"三维度及风格双维度研究框架

## 一、大势研判三维度框架

### 维度一：盈利周期（分子端）
- **核心逻辑**：企业盈利是股价的最终驱动力
- **关键指标**：
  - A股全市场净利润增速（累计同比）
  - 工业企业利润增速
  - PPI同比变化（价格因素）
  - 库存周期位置
  - 产能周期拐点

### 维度二：估值资金（分母端）
- **核心逻辑**：估值扩张 + 资金流入 = 行情放大器
- **估值维度**：
  - 全市场PE/PB历史分位数
  - 股债收益率对比（EY-BD）
  - 风险溢价水平
- **资金维度**：
  - 北向资金净流入
  - 两融余额变化
  - ETF净申购
  - 主动偏股基金发行

### 维度三：多周期共振
- **三周期框架**（华泰2025年度策略）：
  1. **库存周期**：财政脉冲→信用修复→库存周期启动
  2. **产能周期**：供给侧改革效果显现，产能利用率回升
  3. **地产建筑周期**：地产销售企稳→产业链修复
- **共振信号**：三周期同时向上拐头 = 趋势性机会

## 二、风格研判双维度框架

### 维度一：市值风格（大盘/小盘）
- **驱动因素**：经济周期位置、流动性环境
- **规律**：
  - 经济复苏前期：大盘占优（龙头确定性溢价）
  - 经济复苏中后期：小盘占优（弹性更大）
  - 流动性宽松：小盘占优

### 维度二：估值风格（成长/价值）
- **驱动因素**：无风险利率、企业盈利结构
- **规律**：
  - 利率下行：成长占优
  - 利率上行：价值占优
  - 盈利弹性大：成长占优
  - 盈利稳定防御：价值占优


## 四、核心信号体系

### 做多信号（红旗飘飘）
- [ ] 盈利周期：PPI环比转正，工业企业利润增速回升
- [ ] 估值周期：全A PE处于历史30%分位以下
- [ ] 资金周期：北向资金连续净流入
- [ ] 三周期共振：库存+产能+地产同时向上

### 做空信号（绿旗飘飘）
- [ ] 盈利周期：PMI跌破荣枯线，盈利增速持续下行
- [ ] 估值周期：全A PE处于历史70%分位以上
- [ ] 资金周期：两融余额快速萎缩
- [ ] 三周期背离：周期信号相互矛盾

## 五、节奏判断

- **上半年**：压力期，防御为主（哑铃策略）
- **下半年**：机会期，进攻为主（顺周期）
- **核心矛盾**：关注地产销售企稳和产能周期拐点

## 六、数据更新频率

- **日度**：指数行情、资金流向
- **周度**：板块轮动、风格信号
- **月度**：盈利预测、估值分位
- **季度**：周期拐点研判
